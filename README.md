# VIT SMART

## Project Overview

VIT SMART is a student-level prototype for EMG-based muscle fatigue assessment and exercise support.  
The system collects EMG signals, displays real-time waveform data, provides simple fatigue-state feedback, and supports users with AI-assisted guidance.

The project focuses on helping gym users, amateur athletes, and rehabilitation users understand muscle condition more clearly during exercise.

## Current Prototype

The current prototype includes:

- EMG signal collection
- Real-time EMG waveform visualisation
- Short-window fatigue assessment
- Three-state feedback:
  - Normal
  - High Load
  - Fatigue Risk
- Pose analysis and posture correction support
- AI assistant guidance
- Local dashboard interface
- User validation evidence

## Key Design Decision

Based on user testing, we simplified fatigue feedback from raw numerical EMG values into three clear states.

Users found numerical values difficult to interpret, while the three-state feedback helped them understand their muscle condition more quickly and confidently.
