# VIT SMART

## Project Overview
VIT SMART is a student-level prototype for EMG-based muscle fatigue monitoring.  
It focuses on collecting and interpreting muscle electrical signals to help detect muscle fatigue earlier, reduce injury risk, and support safer training.

## Sprint 1 Goal
By the end of Week 5, a real user can complete the basic VIT SMART monitoring flow, and the system can collect EMG signals, show muscle status, and trigger a simple warning.

## Must-have Features
1. EMG signal collection  
2. Basic muscle fatigue monitoring  
3. Fatigue / overload warning feedback
